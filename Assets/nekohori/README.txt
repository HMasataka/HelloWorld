■ 実行環境
    Unity ver. 2022.3.6f1 以上
    VRChat SDK - Base 3.5.2 以上
    VRChat SDK - World 3.5.2 以上

■ 事前にインストールが必要なシェーダー/アセット
    なし

■ 説明
    入力した内容が他人と同期されるInputfieldです。
    URLの共有や、Streamkeyの共有が外部のテキストチャットを介さずに行えます。

■ 設置方法
    シーンに Asset/nekohori/SyncedText の「SyncedTextBox」プレハブを配置してください。
    基本的にポン置きで使えます。
